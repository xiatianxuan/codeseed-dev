什么是 Flask？

Flask 是一个用 Python 编写的轻量级 Web 应用框架。 它被称为“微框架”，因为它简单、灵活，且不强制使用特定的工具或库。 你可以用几行代码就启动一个 Web 服务器！

为什么选择 Flask？

  • ✅ 极简设计，学习曲线平缓
  • ✅ 适合小型项目、API 服务、原型开发
  • ✅ 社区活跃，插件丰富（如 Flask-SQLAlchemy、Flask-Login）
  • ✅ 与 Python 生态无缝集成

快速开始：Hello World

安装 Flask（需先安装 Python）：

pip install Flask

创建 app.py：

from flask import Flask

app = Flask(__name__)

@app.route("/")
def hello():
    return "<h1>Hello, Flask!</h1>"

if __name__ == "__main__":
    app.run(debug=True)

运行：

python app.py

访问 http://localhost:5000，你将看到 "Hello, Flask!"。

Flask vs 其他框架

与 Django（“全栈框架”）不同，Flask 不提供 ORM、用户认证等内置功能， 而是让你自由选择组件。这使得 Flask 更适合：

  • 学习 Web 开发原理
  • 构建 RESTful API
  • 需要高度定制化的项目

结语

Flask 是进入 Python Web 开发世界的绝佳起点。 无论你是学生、竞赛选手（如 CSP-J/S），还是 hobbyist， 都能用它快速实现想法。
